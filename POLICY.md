# 「조상이 도왔다」 운영 원칙 (POLICY) — 코드와 함께 승계

> 운영 주체가 누구로 정해지든 이 원칙은 그대로 승계된다. 특정 소유자에 종속되지 않는다.

## A. 편파 방지 (중립성) — 절대 규칙
- **R1. 가문에 높낮이 없음.** 모든 성씨·본관은 동등. 우열·서열로 표현하지 않는다.
- **R2. 우열어 금지.** 특정 성씨·본관을 더 높거나 낮게 보이게 하는 표현은 빌드 게이트에서 차단한다. 금지 목록은 코드 게이트가 관리하고 문서에는 반복 기재하지 않는다.
- **R3. 인구는 가치가 아니라 맥락.** 인구순 정렬·홍보성 비교 금지(가나다·권역 균등). 수치는 "통계청 2015 약 N명(추정)"으로만.
- **R4. 자동 본관 배정 금지.** "잘 모르겠어요"를 규모가 큰 본관에 자동 편입하지 않는다(추측·쏠림 방지).
- **R5. 분쟁·복수설 중립 병기.** 시조·계통이 갈리면 "~로 전한다"로 병기. 단독 종친회 주장 미채택.
- **R6. 불확실성 표기.** verifyLevel != verified면 단정 어미 차단(전승·확인 필요 표기).
- **R7. 포용 우선.** 족보 없는 사람(귀화·다문화·입양·희성·본관모름)도 동등 참여(이름 길). 아직 기록에 없는 이름도 안내한다.
- **R8. 지역 균형.** 특정 지역을 우위로 비교하는 표현 금지.

## B. 주체(主體) 미정 — 무료·무계정 원칙
- **익명·무계정 기본.** 이용자는 로그인 없이 성씨·본관·이름 기록을 볼 수 있다. 테스트 회원 기능은 서버 전송 없는 브라우저 저장만 허용한다.
- **가계 증명 금지.** 앱의 성씨·본관 안내는 문화관광 정보이며 족보 인증, 신분 확인, 가족관계 확인, 특정 가계 확정에 쓰지 않는다.
- 운영·결제·법인 '주체'가 미정인 동안: **유료·계정 종속 결정 전면 보류.**
- 지금 기술 명의(개인 임시)는 **사업 주체가 아니다.** 자산 원본은 WORKSPACE·G드라이브 중립 보관.
- **주체 확정 후로 미루는 것**: 네이버 지도(NCP 계정·카드·과금)·VWorld 키·정식 도메인·분석(GA4/네이버)·수익화·공식 데이터 제휴·상표 출원.
- **주체 결정 트리거(하나라도 닿으면 결정 강제)**: 수익기능 ON 직전 / 개인정보 정식수집 / 정식도메인·네이버 전환 / 공식제휴 / 앱스토어 출시 / 외부투자 / 상표 선점위협 / 트래픽·매출 임계.
- 주체 1순위 현실안(권고): 무료 MVP → 정부 공공·문화 지원사업으로 실적·자금 → 요건 역산. 합법화는 기존 사업자 **와이제이파트너스(365-35-00893)** 업종추가(홈택스 무료)로 즉시 가능(단 중립선언·면책 선행으로 본업 리스크 차단).

## C. 지도
- **현재 = OSM 표준 타일(무료·무키, 한글 라벨).** attribution "© OpenStreetMap contributors" 필수.
- OSM 재단 타일서버는 소규모만 허용 → 트래픽 증가/주체 확정 시 자체 호스팅·VWorld·네이버로 전환.
- 지도 호출은 한 함수(`buildMap`)에 격리 → 전환은 타일 교체로 끝남(추후 map_adapter.js로 더 분리).

## D. 기본값
- 익명·무회원 이용 기본(개인정보 최소). 면책·정정창구 내장. 어떤 유료 등록도 트리거 전 금지.

## E. 지역 활성화·광고·제휴 수익화 가드레일
- **민간 플랫폼 표현.** 공식 협약·공문·승인번호 전에는 정부·지자체 공식 앱, 인증 플랫폼, 후원 명칭, 기관 로고를 쓰지 않는다.
- **성과 보장 금지.** 지역 방문, 매출, 광고효과, 지원사업 선정, 보조금 수급 가능성을 보장하지 않는다. 수치는 예측·실험·리포트 기준으로만 분리한다.
- **광고·협찬 표시.** 유료 노출, 제휴, 협찬, 쿠폰, 예약 연계는 모든 화면에서 광고·협찬·제휴 라벨을 고정 표시하고 일반 추천처럼 보이게 하지 않는다.
- **문화정보와 광고 분리.** 성씨·본관·성씨·연원 기록은 문화관광 정보로 두고, 지역 협력 스팟은 별도 영역에 배치한다.
- **개인정보 목적 분리.** 성씨, 본관, 가족기록, 위치, 방문이력, 광고 타깃팅을 묶어 쓰지 않는다. 정식 수집 전에는 닉네임·데모 이메일 수준의 테스트만 허용한다.
- **공공데이터 라이선스 확인.** 관광공사·행안부·공공데이터·공공저작물은 출처, 갱신일, 이용조건, 상업 이용 가능 여부를 데이터별로 남긴다.
